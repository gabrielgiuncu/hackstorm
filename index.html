HOW TO PLAY
1
Install Python
You need Python 3.7+ installed. No extra libraries needed!
# Windows: download from https://python.org
# Mac: brew install python3
# Linux: sudo apt install python3
# Termux: pkg install python
2
Download HackStorm
Clone the repo or download the ZIP file.
git clone https://github.com/gabrielgiuncu/hackstorm.git
cd hackstorm
3
Play Offline (Single Player)
Full game with all features, saves locally.
python hackstorm.py
4
Play Online (Multiplayer)
Connect to the public server. Chat, compete, climb the leaderboard!
python client.py

# When asked for server address type:
# market-celebration.gl.joinmc.link
5
üì± Play on Android (Termux)
Install Termux from F-Droid (NOT Play Store). Then run these commands:
pkg update && pkg install python git -y
git clone https://github.com/gabrielgiuncu/hackstorm.git
cd hackstorm
python hackstorm.py # offline
python client.py # online
6
Host Your Own Server
Want to run a private server for friends?
python server.py
